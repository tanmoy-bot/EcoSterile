# 🔐 Admin Panel - Quick Reference Card

## 📍 Access Points

```
Login:     /auth/admin-login.html
Dashboard: /auth/admin-dashboard.html
```

## 🔧 Configuration

### Step 1: Update Admin Emails
Edit these 3 files and update the `ADMIN_EMAILS` array:

```
File 1: auth/admin-login.html (line 183)
File 2: auth/admin-dashboard.html (line 200)
File 3: services/admin-utils.js (line 4)
```

**Add your email:**
```javascript
const ADMIN_EMAILS = [
    "admin@ecosterile.com",
    "dev@ecosterile.com",
    "YOUR_EMAIL@example.com"  // ← Add here
];
```

### Step 2: Test It
1. Go to `/auth/admin-login.html`
2. Login with your credentials
3. Export some data
4. Done! ✅

| Issue | Solution |
|-------|----------|

| "Login failed" | Verify credentials in Firebase |
| "Data won't load" | Check Firebase DB & rules |
| "Export failed" | Check browser console, refresh |
