## 🔧 Configuration (3 Steps)

### Step 1: Update Admin Emails
Edit and add your email to `ADMIN_EMAILS` array in:
1. `auth/admin-login.html` (line 183)
2. `auth/admin-dashboard.html` (line 200)
3. `services/admin-utils.js` (line 4)

```javascript
const ADMIN_EMAILS = [
    "admin@ecosterile.com",
    "dev@ecosterile.com",
    "yourname@example.com"  // ← Add your email
];
```

### Step 2: Test
Navigate to `/auth/admin-login.html` and login

### Step 3: Deploy
Push to production when ready
