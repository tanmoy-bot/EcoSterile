🌿 Crop Disease Classifier
Identify common crop diseases quickly using your smartphone camera. This tool runs a lightweight image classifier and returns a probable diagnosis — use it to get fast guidance.
How to use
  1. Click Open Classifier below.
  2. Allow camera access or upload a clear photo of the affected leaf.
  3. Center the diseased area and avoid blurry photos — natural daylight helps.
  4. Wait a few seconds for the model to return the predicted class and confidence.
  5. Follow recommended treatment guidance from trusted agricultural sources.
